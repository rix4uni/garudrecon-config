## garudrecon-config
Helper of garudrecon

## Setup
```
Steps:
Step 1: Fork this repo and make it private.
Step 2: Add you api as much as you can.

Step 3: Now setup config files to you system there are 2 ways you can use any of them.

1. Using HTTPS with Username and Personal Access Token (PAT):
git clone https://YOURUSERNAME:PERSONAL_ACCESS_TOKEN@github.com/YOURUSERNAME/private-config.git --depth 1
cp -r private-config/* ~/.config

OR
2. Via downloading using browser Personal Access Token (PAT) not required:
Download ZIP and unzip
cp -r private-config/* ~/.config
```
